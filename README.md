# ai-unity-avatar
Unity C#で実装され、MCP(Model Context Protocol)を通じてAIエージェントと通信します。  
VRMやLive2Dの3Dアバターをリアルタイムで動かすツールです。  
AIとの対話内容に応じてアバターが表情変化、モーション再生、リップシンクを行い表現を実現します。  
Voicevoxとの統合により音声合成にも対応し、視覚と聴覚の両面からインタラクティブな体験を提供します。  
開発者とAIが協働しながら、日曜プログラミング的なスタイルで機能を拡張していくことを目指しています。
